About PubMed Abstract Scraper

What is this?

This web application allows you to search PubMed, the premier database of biomedical literature, and extract article abstracts in bulk. It uses the official NCBI E-utilities API to provide fast, reliable access to millions of research articles.

How to Use

  1. Enter your search query

    Type any medical or scientific term. Use AND, OR, NOT operators for complex searches.

  2. Set the number of results

    Choose how many articles to retrieve (1-500).

  3. Click Search

    The app will fetch articles from PubMed and display full abstracts.

  4. Export your data

    Download results in CSV, JSON, or Excel format for further analysis.

Advanced Search Tips

Phrase Search

"lung cancer"

Use quotes to search for exact phrases

Boolean Operators

cancer AND treatment

Combine terms with AND, OR, NOT

Field Tags

smith[Author]

Search specific fields: [Title], [Author], [Journal]

Date Filters

2020:2023[pdat]

Filter by publication date range

API Access

You can also access this scraper programmatically using our REST API.

Example API Request:

POST /api/search
Content-Type: application/json

{
  "query": "lung cancer",
  "max_results": 20
}

Example Response:

{
  "query": "lung cancer",
  "count": 20,
  "results": [
    {
      "pmid": "12345678",
      "title": "Article Title",
      "abstract": "Abstract text...",
      "authors": "John Doe, Jane Smith",
      "journal": "Nature",
      "publication_date": "2023 Jan",
      "url": "https://pubmed.ncbi.nlm.nih.gov/12345678/"
    }
  ]
}
