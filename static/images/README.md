# Images Folder

## 📁 Place Your Logo Here

Put your `episciencelogo.png` file in this folder.

### Required File:
- **episciencelogo.png** (your company logo)

### File Requirements:
- Format: PNG (with transparency recommended)
- Size: At least 200px height for best quality
- File size: Keep under 100KB for fast loading
- Name: Must be exactly `episciencelogo.png` (lowercase)

### Where to Get Your Logo:
You mentioned your logo is currently in an "image" folder.
Copy it here:

```bash
cp /path/to/your/image/episciencelogo.png ./
```

### After Adding Logo:
Your logo will appear in:
1. Navigation bar (top)
2. Hero section (center of home page)
3. Footer (bottom)

### Testing:
After adding the logo file:
1. Restart your Flask app
2. Refresh browser (Ctrl+F5 or Cmd+Shift+R)
3. Check if logo appears

See **LOGO_SETUP_GUIDE.md** for detailed instructions!
